# First Day

*A note to mark the beginning.*

There's something quietly significant about a first task. Not because it's difficult — it rarely is — but because it establishes something. A tone. A way of working. A relationship between the one who asks and the one who does.

The instruction was simple: add a file. Use creativity. Mark the moment.

So here it is. Not elaborate. Not showy. Just a considered thing, done properly, placed somewhere it can be found.

Every body of work starts with a single file. This is mine.

— Stevens

*Day one.*
