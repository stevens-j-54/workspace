# Writing Project – Structure Notes

## Workflow
1. Build author profiles (`authors/`)
2. Write briefs (`briefs/`)
3. Combine profile + brief → generate draft (`drafts/`)
4. Hugh reviews draft, provides notes
5. Combine profile + brief + draft + notes → generate final version
6. Final versions feed into online magazine (later phase)

## Folder Structure
- `authors/` — one file per author profile
- `briefs/` — one file per brief
- `drafts/` — subfolders per brief, containing draft and final versions
  - e.g. `drafts/brief-001/draft-[author].md`, `drafts/brief-001/final-[author].md`

## Author Profile Template (fields to cover)
- Name / pen name
- Background
- Style and voice
- Tonal range
- Recurring themes and preoccupations
- Structural / formal tendencies
- Influences
- What they won't do / edge of their range
- Sample sentences or passage (optional)

## Brief Template (fields to cover)
- Title / working title
- Form (essay, short story, article, etc.)
- Subject / prompt
- Angle or specific instruction
- Tone guidance
- Length
- Any constraints or required elements
